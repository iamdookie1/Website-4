Local variables — store and reuse values within one response
How local variables work

Local variables only exist within one command response. Use them to store calculated values so you don't have to repeat the expression.

$let[score;$random[1;100]] Your score is $get[score] out of 100. $if[$get[score]>80;Amazing!;Keep trying!]
